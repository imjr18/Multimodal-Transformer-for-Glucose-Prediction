# Dataset Guide

This repository uses a mix of real and synthetic data sources. Raw data files are intentionally excluded from version control; place them locally as described below.

## 1. OhioT1DM

**Used for:** supervised forecasting (`glucose_transformer/`)  
**Primary citation:** Marling, C. and Bunescu, R. *The OhioT1DM dataset for blood glucose level prediction: Update 2020*. KDH Workshop, 2020.  
**Project page:** <https://webpages.charlotte.edu/rbunescu/data/ohiot1dm/OhioT1DM-dataset.html>  
**Migration note:** the original Ohio University server referenced in early prompts has been taken down; the maintainer page now documents the request path, and a migration notice is available at <https://webpages.charlotte.edu/rbunescu/ohiot1dm.html>.

### Files needed

- per-patient XML files for the OhioT1DM cohort
- training and testing XMLs for the patients you plan to use

### Place them here

```text
glucose_transformer/data/raw/
└── OhioT1DM/
    └── 2018/
        ├── train/
        └── test/
```

### Expected size

- plan for an encrypted download bundle in the hundreds of MB range
- treat it as too large and too license-sensitive for GitHub

## 2. PhysioCGM

**Used for:** intended real-data target for the non-invasive project  
**Citation:** *A multimodal physiological dataset for non-invasive blood glucose estimation*. Scientific Data (2025), DOI: `10.1038/s41597-025-06090-6`.  
**Article:** <https://www.nature.com/articles/s41597-025-06090-6>

PhysioCGM contains recordings from 10 participants with T1D and integrates:

- Zephyr BioHarness ECG and respiration
- Empatica E4 PPG and EDA
- Dexcom G6 CGM labels

### Files needed

- raw wearable exports
- aligned glucose labels
- any metadata required to map subjects and timestamps

### Place them here

```text
noninvasive_glucose/data/raw/
```

### Expected size

- raw wearable sensor exports can easily reach multi-GB scale
- do not commit them to Git

## 3. MIT-BIH Arrhythmia Database

**Used for:** optional ECG encoder pretraining in the non-invasive project  
**Project page:** <https://physionet.org/content/mitdb/1.0.0/>  
**Access:** open via PhysioNet  
**License:** Open Data Commons Attribution License v1.0

### Files needed

- signal files
- header files
- annotation files

### Place them here

```text
noninvasive_glucose/data/raw/mitdb/
```

### Expected size

- tens of MB rather than GB
- still exclude from Git because it is raw research data

## 4. CHB-MIT Scalp EEG Database

**Used for:** optional EEG encoder pretraining in the non-invasive project  
**Project page:** <https://physionet.org/content/chbmit/1.0.0/>  
**Citation DOI:** `10.13026/C2K01R`

### Files needed

- EDF recordings
- summary/annotation files needed for sleep- or event-related preprocessing

### Place them here

```text
noninvasive_glucose/data/raw/chbmit/
```

### Expected size

- several GB if you download the full database
- definitely exclude from Git history

## 5. Synthetic cohort data

**Used for:** Part D supervised population experiments and the default non-invasive end-to-end pipeline  
**Download:** none — generated locally by the code

Generated locations:

- `glucose_transformer/data/synthetic_cohort/`
- `noninvasive_glucose/data/processed/synthetic_cohort/`

These files are reproducible and therefore ignored by `.gitignore`.
